FEATURES OF HTTP

1. CONNECTIONLESS PROTOCOL

In http protocol after sending request to http client,connection betwn client and server is disconnected and then clent waits for response.
Http server processes the request, send response for that connection and connection is re-estblished between http client and server.

2. Media dependent independent

HTTP handles any type of data, both (client & server) know how to handle data.

3. Stateless protocol

HTTP (client & server) known to each other during current http request & after processing current reques , both of them forget about each other.
